## Keyboard shortcuts

| Description     | Shortcut             |
|-----------------|----------------------|
| Command palette | Ctrl (⌘) + Shift + P |
| Intellisense    | Ctrl (⌘) + Space     |
| Multi cursor    | Alt + Left Click     |
| Open file       | Ctrl (⌘) + P         |
| Refactor        | Ctrl + Shift + R     |
| Rename symbol   | F2                   |
